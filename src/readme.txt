sit_centre monitoring

список используемых библиотек можно просмотреть в файле package.json

работа с состоянием данных производится через reduxjs@/toolkit
асинхронные запросы через axios 

последовательность импорта:
    --библиотеки
    --файлы логики
    --компоненты
    --стили и картинки



навигация по роутам
    --routes


основная логика реализации проекта прописана в папке
    --src
        --features


компоненты UI в папке
    --src
        --page
        --components


папка features
    асинхронные запросы реализуются через thunk middleware
    логика запросов прописывается в extrareducers
    дополнительные операции с состоянием данных производится в reducers

    получить доступ к состоянию данных в компонентах можно осуществить через импорт useSelector
    реализовать reducer в компонентах можно через импорт useDispatch
